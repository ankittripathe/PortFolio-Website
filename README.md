## 📦 Getting Started

### Prerequisites

- Node.js (v14 or above)
- npm or yarn

## 🚀 How to Run the Project

```bash
# 1. Clone the repository
git clone https://github.com/ankittripathe/TrendWear.git
cd TrendWear

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```
